This is the hard-drive. It stores everything with a disc and needle. The disc spins while the needle reads it.

This card is like an extension. There are some slots on the motherboard. The PCU card is where the ports come from.

This is the power supply. It powers pretty much everything in the computer. It is safe if it has a cover on it. If the cover is not on, it is very dangerous! Don't touch anything if it is off!

This is a graphics card. It sends video data to your monitor. There are some graphics cards that make everything HD.

This is the motherboard. It keeps everything in check. It connects the power supply, the harddrive, etc. to everything.
